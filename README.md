# DataEx — Meta-Analysis Data Extractor

RCT（ランダム化比較試験）論文から、システマティックレビュー／メタ分析に必要な数値データを自動抽出するブラウザ単体で動作するツールです。PDFをアップロードしてAPIキーを入力するだけで、症例数・イベント数・平均値・SD・効果推定値などをアウトカム毎に構造化抽出します。

## 特徴

- **単一HTMLファイル**でブラウザ完結。サーバ不要。
- **マルチAIプロバイダー対応**: Claude (Anthropic) / Gemini (Google) / OpenAI から選択・組み合わせ可能
- **4段階抽出パイプライン**（Ultraモード）: 並列抽出 → 結果統合 → Critic監査 → 個別検証
- **ITT / PP 集団の分離抽出**、3群比較、時点別・サブグループ別も自動分割
- **揺らぎの完全保持**: Abstract と本文の数値差異、術前・術後・変化量を個別に保持
- **信頼度スコア**付与、**根拠原文（evidence text）** を各項目に紐付け
- **PDFビュアー統合**で該当箇所を即座にハイライト可能
- TSV / CSV エクスポート対応

## 使い方

1. このリポジトリから `meta-analysis-extractor v3-opus47.html` をダウンロード
2. ブラウザで開く（ダブルクリック、または`file://`で開く）
3. 使用したい AI プロバイダーを有効化し、APIキーを入力
4. 抽出精度モードを選択（標準 / 高精度 / Ultra）
5. PICO を入力、PDFをドラッグ＆ドロップ、「抽出実行」

APIキーの取得方法は、アプリ内の「APIキーの取得方法」セクションに詳細ガイドがあります。

## 精度モード

- **標準**: 単一抽出、コスト最小
- **高精度**: アンサンブル抽出 + Critic監査
- **Ultra**: 4段階パイプライン（Critic + 個別検証まで実施）

Critic 役は有効なプロバイダーから最精度のモデルが自動選択されます。Claude Opus 4.7 があればそれが優先されますが、無くても Gemini 2.5 Pro や GPT-5 で動作します。

## セキュリティ

- APIキーはお使いのブラウザの `localStorage` にのみ保存されます（任意）
- 外部送信は各公式APIへの正規リクエストのみ
- セッション保存時のJSONにはAPIキーは含まれません（`***REDACTED***` に置換）

## 免責事項

- 本ツールは強力なアシスタントですが、抽出結果は**必ず原論文と照合してください**
- 特に **SD と SE の混同**、図表からの読み取り誤差に注意
- 医学的な最終判断は研究者ご自身で行ってください
